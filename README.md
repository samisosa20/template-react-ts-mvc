using architecture MVC and TailWind library

## Requirements
- node version >= 16.15.x
- npm version >= 8.5.x


## How to install
`git clone https://github.com/samisosa20/template-react-ts-mvc.git`

`cd template-react-ts-mvc`

`npm i`
